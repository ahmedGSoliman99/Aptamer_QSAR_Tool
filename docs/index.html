Aptamer QSAR Tool
RNA/DNA aptamer modeling

Interaction-aware aptamer QSAR.

A scientific local web app for aptamer sequence descriptors, user-entered aptamer-target interaction features, model training, prediction, virtual aptamer design, ranking, visualization, and report export.

Sequence descriptors

GC content, base composition, k-mer frequencies, entropy, molecular-weight proxy, melting-temperature approximations, G-rich motifs, and self-complementarity.

Interaction descriptors

Hydrogen bonds, hydrophobic contacts, pi-stacking, salt bridges, metal coordination, base pairing, base stacking, water bridges, and target contacts.

QSAR models

Regression and classification workflows for affinity, Kd, IC50, docking score, activity classes, toxicity, and other aptamer endpoints.

New aptamer design

Generates mutated aptamer candidates from a selected seed, predicts their expected score with the trained model, and ranks candidates predicted to improve over the seed.

Scientific Basis

The tool converts RNA/DNA aptamer sequences and curated interaction information into numerical descriptors, then uses machine-learning models to relate those descriptors to biological or computational endpoints.
